About

This playish plugin is for handling the communication between the browser and your unity project. It takes care of input sent from controllers that are handled by the playish platform. With this plugin you can then use input from controllers from playish including dynamic mobile controllers. It can also be used to send information and actions to the server for example change what dynamic controller that should be used.
